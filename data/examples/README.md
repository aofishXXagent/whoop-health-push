# Example Data

These are **simulated** data files showing the format exported by this tool.
All values are fictional — they don't represent any real person's health data.

Use these to understand the data schema before connecting your own WHOOP account.

## Files

| File | Description |
|------|-------------|
| `recovery.csv` | Daily recovery score, HRV, resting HR, SpO2, skin temp |
| `sleep.csv` | Sleep duration, stages (deep/REM/light), efficiency, disturbances |
| `cycles.csv` | Daily strain, heart rate, energy expenditure |
| `workouts.csv` | Individual workout records with HR zones and distance |

## Key Metrics

- **Recovery Score** (0-100%): Green ≥67%, Yellow 34-66%, Red <34%
- **HRV** (ms): Heart rate variability — higher = better recovery
- **Strain** (0-21): Daily exertion level — match to recovery for balance
- **Sleep Performance** (%): Actual sleep vs. body's need
